# Divine Private Home - Student Result Portal

A responsive **student result web application** for Divine Private Home school, featuring secure **roll number & password login**, **Top-3 student highlights with badges**, and a modern **white + gradient background design**.

---

## Features

- Secure result access using **Roll Number + Password**  
- Displays **Top 3 students of the week** with **Gold, Silver, Bronze badges**  
- Clear **grade calculation** and **total marks display**  
- Modern, responsive design with **white + blue & purple gradient background**  
- Easy to customize for other schools

---

## Technologies Used

- HTML5  
- CSS3 (Flexbox & Gradient)  
- JavaScript (for result logic)
